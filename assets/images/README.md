# Visual Assets for Alien Invasion Pinball

This directory will contain all the visual assets for the game. Below is a list of the required assets and their status.

## Background
- `background.png` - Pinball table background with space theme, synthwave colors, stars, and Earth

## Game Elements
- `flipper_left.png` - Left flipper sprite sheet
- `flipper_right.png` - Right flipper sprite sheet
- `ball.png` - Pinball ball with glowing cyan color
- `alien_scouts.png` - Sprite sheet for small UFO-shaped bumpers
- `alien_fighters.png` - Sprite sheet for alien-shaped drop targets
- `asteroids.png` - Sprite sheet for asteroid field rollover lanes
- `mothership.png` - Sprite sheet for the alien mothership (multiball trigger)
- `ramps.png` - Sprite sheet for defense ramps

## UI Elements
- `logo.png` - Game logo in synthwave style
- `score_digits.png` - Sprite sheet for score display digits
- `life_icon.png` - Earth icon for life counter
- `game_over.png` - Game over screen
- `start_button.png` - Start button

## Asset Generation Process

All assets will be generated using AI image generation tools with the prompts specified in the asset-generation-guide.txt file. The process involves:

1. Generate initial assets using AI tools
2. Select the best results
3. Perform minor cleanup/resizing if needed
4. Ensure consistent style across all assets
5. Optimize for web use

## Style Guidelines
- 8-bit pixel art style
- Synthwave color palette (purples, pinks, cyans)
- Consistent pixel resolution (32x32 grid for smaller elements)
- Transparent backgrounds where appropriate
